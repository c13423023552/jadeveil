# Changelog

## 3.0.0 — 2026-07-12

首个公开版本。

- **清透玻璃**：chrome 层（侧栏/Ribbon/状态栏）真实 backdrop-filter 玻璃，浓度双滑杆（浅 0.64 / 深 0.85 默认）；正文纸面恒 ≥0.96 不透明，滚动路径零模糊层
- **青玉 accent 体系**：色相 166°、饱和度 46%；checkbox / tag / 选区 / 焦点光环 / callout 全族派生，色相滑杆全局换装不出霓虹
- **暖纸基座**：浅色暖象牙纸 + 全灰阶暖调；深色墨玉石墨，侧栏与纸面同明度带
- **中文排版**：text-autospace / text-spacing-trim 渐进增强、标题 auto-phrase 断行、42em 行宽、代码 CJK 行高 1.7、连字禁用
- **可访问性**：双模式正文 13.0 / 11.4:1；链接、语法高亮板、callout 题字、任务态 glyph、焦点指示全部 AA 达标（独立评审实算校准）
- **语义任务状态**：`[-]` 取消 / `[>]` 顺延 / `[?]` 疑问 独立 glyph 与配色
- **变体**：OLED 纯黑、纸感加深、全玻璃关闭逃生门，均有运行时冒烟断言覆盖
- **打印**：@media print 白纸黑字基线
- **工程**：token 公共/内部分级契约、!important 台账、backdrop-filter 白名单、依赖台账 D1–D6、lint（CI）+ smoke（真机）双工具
